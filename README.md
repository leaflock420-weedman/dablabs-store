Upload `data/shopify-import.csv` → **Products → Import**

## Go live on Shopify

1. Create store at [shopify.com/au](https://www.shopify.com/au) — **Dab Labs**, Australia, AUD
2. Upload theme: **Online Store → Themes → Add theme → Upload zip file** → use `dablabs-theme.zip`
3. Upload logo + favicon from `brand/` folder in theme customizer
4. Enable payments: Shopify Payments + Afterpay + Zip (Settings → Payments)
5. Set shipping: AU standard $9.95, free over $100, express $14.95
6. Point `dablabs.com.au` domain from Wix to Shopify (Settings → Domains)

Full checklist: `config/SETUP-CHECKLIST.md`
